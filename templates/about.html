About Edit Ninja

Edit Ninja helps you edit your images easily. This web-app is created using Python Flask and OpenCV.
It can perform the following processes on a image with the help of OpenCV :

  • Convert to JPG
  • Convert to PNG
  • Convert to Grayscale
  • Crop image
  • Resize image to 350*350 px
  • Resize image to 1000*1000 px
  • Rotate image vertically
  • Rotate image horizontally
  • Apply Gaussian Blur
  • Apply Median Blur
  • Apply Bilateral Filter
  • Apply Border
